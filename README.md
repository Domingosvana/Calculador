# 🧮 Calculator App

A simple Android calculator application built to perform basic arithmetic operations with a clean and modern interface.

This project was created to practice **Jetpack Compose UI development, state handling, and building interactive components** in Android.

---

## ✨ Features

- Addition
- Subtraction
- Multiplication
- Division
- Percentage calculation
- Light and Dark theme toggle
- Simple and intuitive user interface

---

## 🛠 Tech Stack

- **Kotlin**
- **Jetpack Compose**
- **Material 3**
- **State Management with Compose**

---

## 🏗 Project Structure

The project follows a simple structure to keep the code organized and easy to understand:

```
feature
model
screens
components
theme
```

This lightweight structure focuses on clarity and simplicity rather than complex architectural patterns.
